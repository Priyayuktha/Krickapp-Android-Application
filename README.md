A lightweight **Android application** for managing and displaying local
cricket match data such as college tournaments and inter-community
games.

Unlike mainstream apps (e.g., Cricbuzz) that rely on third-party APIs,
**KrickApp** uses a **manual but real-time scoring system** through an
**Admin Panel (Scorer's Desk)**. This makes it ideal for small-scale
matches where digital coverage is usually missing.
