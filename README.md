# ğŸ“± KrickApp

A lightweight **Android application** for managing and displaying local
cricket match data such as college tournaments and inter-community
games.

Unlike mainstream apps (e.g., Cricbuzz) that rely on third-party APIs,
**KrickApp** uses a **manual but real-time scoring system** through an
**Admin Panel (Scorer's Desk)**. This makes it ideal for small-scale
matches where digital coverage is usually missing.

## ğŸš€ Features

-   **Manual Match Scoring** -- Venue, toss result, squads, runs,
    wickets, overs, highlights\
-   **Live Updates** -- Instantly visible to all users\
-   **Match Management** -- Create, schedule, and update matches\
-   **Scorecard** -- Batting & bowling stats, over summaries,
    highlights\
-   **Lightweight & Fast** -- Optimized for low-end devices and minimal
    internet usage\
-   **Ad-Free Experience** -- Clean and distraction-free UI

## ğŸ— Modules

1.  **Home**
    -   List of matches (Scheduled, Ongoing, Completed)\
    -   Match cards showing team names, date/time, live score
2.  **Scorer's Desk (Admin Panel)**
    -   Create matches & squads\
    -   Manage toss results\
    -   Update scores (ball-by-ball / over-by-over)\
    -   Add highlights
3.  **Matches**
    -   Match info (venue, date, toss result, type)\
    -   Squads with roles\
    -   Full scorecard\
    -   Overs summary & highlights
4.  **More**
    -   User profile & settings\
    -   About, Contact, Feedback\
    -   Terms & Conditions
